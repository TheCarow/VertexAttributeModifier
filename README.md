# Vertex Attribute Modifier
Unity Engine tool for modifying the vertex attributes of a mesh.

![VAM](https://github.com/user-attachments/assets/92c90bf5-abc6-41ce-88c0-1434e5969e6e)

## Features
* Vertex attribute overriding (format and dimension).
* Vertex attribute stripping.
* Modification rules can target multiple folders.
* Uses AssetPostprocessor to apply modifications.

## Usage
1. Open **Tools > Vertex Attribute Modifier**
2. Select Add Rule.
3. Configure the rule by specifying a Source Directory and vertex attribute modifications.
